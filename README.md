# Academia Maestro - Sistema de Gestão e Análise

## Descrição do Projeto

Este projeto consiste em um sistema de gestão e análise de dados para a Academia Maestro, focada em oferecer aulas individuais de instrumentos musicais como violão, teclado, guitarra, bateria e violino. O objetivo é organizar informações de alunos, professores, agendamentos, matrículas e pagamentos, além de fornecer dashboards e relatórios para auxiliar na tomada de decisões.

O sistema foi arquitetado com uma estrutura modular, utilizando Python para a lógica de negócio, processamento de dados e dashboards, e SQLite para o armazenamento de dados.

## Funcionalidades Principais

*   **Gestão de Alunos:** Cadastro e acompanhamento de alunos.
*   **Gestão de Professores:** Cadastro e acompanhamento de professores.
*   **Gestão de Aulas:** Registro de aulas ofertadas e agendamento de aulas individuais.
*   **Gestão de Matrículas:** Registro das matrículas dos alunos em aulas específicas.
*   **Controle Financeiro:** Registro de pagamentos e acompanhamento do faturamento.
*   **Análise de Dados:** Scripts para análises de alunos, aulas, professores e financeiro.
*   **Dashboards Interativos:** Visualizações de KPIs e relatórios através de uma interface web.
